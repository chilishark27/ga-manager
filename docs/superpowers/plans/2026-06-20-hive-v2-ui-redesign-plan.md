# Hive v2 UI Redesign Plan

## Goal
为 Hive v2 设计一套独立的任务面板风视觉语言（类似 Linear Issues），区别于其他页面，作为核心功能突出展示。

## Design Direction
- **任务面板风**：卡片式布局，强状态色块，类似 Linear
- **全新设计语言**：Hive 有独立视觉体系，不完全跟随 Sakura 主题
- 深色背景但高对比度，状态用鲜明颜色区分

## Visual Language

### 色彩系统（Hive 专属）
- 背景：`#0d1117`（比 Sakura Night 更深更纯的深色）
- 卡片：`#161b22`（GitHub Dark 风格）
- 边框：`rgba(48, 54, 61, 0.8)`
- 状态色：
  - Running: `#58a6ff`（蓝色脉冲）
  - Done: `#3fb950`（绿色）
  - Pending: `#8b949e`（灰色）
  - Failed: `#f85149`（红色）
  - Blocked: `#6e7681`（暗灰）
- 强调色：`#79c0ff`（天蓝）
- 文字：`#f0f6fc` / `#8b949e` / `#484f58`

### 组件风格
- 卡片：无圆角或极小圆角（4px），左侧状态色条
- 按钮：扁平，hover 有微光效果
- 新建弹窗：全屏居中，毛玻璃遮罩，内部深色卡片
- 进度：彩色进度条替代文字比例

## Page Redesign

### 1. 项目列表页（HivePage）
```
┌──────────────────────────────────────────────────┐
│ HIVE                                    [+ New]   │
│ ─────────────────────────────────────────────────│
│ ┌─── Active ──────────────────────────────────┐  │
│ │ ██ 支付系统接入          4/6 ████████░░ 35m  │  │
│ │ ██ 用户文档生成          2/5 ███░░░░░░ 12m   │  │
│ └────────────────────────────────────────────┘  │
│ ┌─── Completed ───────────────────────────────┐  │
│ │    API 文档生成          3/3 ██████████ 15m   │  │
│ └────────────────────────────────────────────┘  │
│                                                   │
│ Workers: ●●●○○ 3/5 busy                          │
└──────────────────────────────────────────────────┘
```
- 分组：Active / Completed / Failed
- 每个项目卡左侧有 4px 状态色条
- 进度用实心色条而非文字

### 2. 新建项目弹窗
- 深色卡片 + 毛玻璃背景
- 表单字段用浮动 label（Material Design 风格）
- 模板选择用可视化卡片网格，不是下拉框
- 创建按钮用蓝色强调色

│ ← 支付系统接入     running    4/6 ████░░ 35m  CC │

│          │ │ > 读取设计...    │  │               │

- 左侧任务列表：竖向时间线风格，圆点+连线表示状态
- 中间详情：当前任务信息 + 实时日志
- 右侧产出：文件列表，可点击预览
- 底部 context bar

## Changes Required

### CSS (global.css)
- 新增 `.hive2-*` 样式类（不影响旧 hive 样式）
- Hive 专属颜色变量
- 任务面板布局样式
- 状态色条、进度条、时间线样式
- 新建弹窗重新设计

### Components to Rewrite
1. `HivePage.tsx` — 分组列表 + 进度条 + 状态色条
2. `HiveProjectPage.tsx` — 时间线任务列表 + 日志面板
3. `NewProjectDialog.tsx` — 模板卡片网格 + 浮动 label
4. `TaskList.tsx` — 时间线风格竖列
5. `TaskDetail.tsx` — 卡片式详情 + 日志区
6. `ArtifactPanel.tsx` — 文件卡片列表
7. `ContextBar.tsx` — 标签式底栏
